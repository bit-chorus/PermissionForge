# 今日校园请假条伪造器

## 描述

它能帮你伪造今日校园的请假条。

*能用的话不妨动动小手，给个⭐支持一下吧。😀*

## 字段说明

**名字**：请假者的姓名。

**辅导员名字**：批准该假条的辅导员姓名。

**时长（小时）**：请假的时长。

**地点**：学校的地址。

**请假原因**：请假的借口。

## 隐私声明

填写的字段仅作假条显示时用。本程序不会上传你的个人信息，请放心使用。
